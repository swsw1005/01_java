String 클래스

String substring (int start)
>> 현재문자열 객체에서 start 위치부터 끝까지 문자열 발췌

String substring (int start,  int end)
>> 문자열 객채에서 star~end까지 문자열 발췌

int indexOf (String s)
>> 현재 문자열에서  s 찾아서 있으면 첫번째 위치(index) 반환

ex)  i = str.indexOf("d");
        System.out.println(i); // 4
